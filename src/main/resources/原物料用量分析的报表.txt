下面sql计算出：品牌，PN，機型，原物料供应商
select rd.brand,rd.PART_NO,pm.SKU_NO,pm.VENDOR_CODE from repair_detail rd 
left join [gssd-gsp_official_web].dbo.prod_master pm     on pm.PART_NO = rd.PART_NO 
group by 
rd.brand,rd.PART_NO,pm.SKU_NO,pm.VENDOR_CODE
order by rd.brand,rd.PART_NO

遍历上一步的brand和PART_NO，拿到 PN維修數量（为原物料數量之和），原物料名稱，原物料料號，原物料數量和比率
select rdr.NEW_PN  ,rdr.NEW_PN_NAME ,count(rdr.NEW_PN) from repair_detail_record rdr 
where rdr.DETAIL_ID in (
select rd.id  from repair_detail rd 
where rd.brand = 'DELL' and rd.PART_NO = 'HMF7C'
)
and  rdr.NEW_PN is not null and rdr.NEW_PN_NAME is not null
group by rdr.NEW_PN  ,rdr.NEW_PN_NAME
